# Jake Silveira | Professional Portfolio

A modern, responsive personal landing page designed to showcase professional accomplishments in additive manufacturing, automation, and web development. This site is optimized for performance and aesthetic appeal, featuring a clean dark-themed UI with smooth scroll-driven animations.

## 🚀 Features

- **Modern UI/UX:** A professional dark theme using the Inter font stack and a consistent grid system.
- **High-Performance Animations:** Utilizes the `IntersectionObserver` API for smooth, performance-friendly section fade-ins.
- **Responsive Design:** Fully fluid layout that adapts seamlessly from mobile devices to large desktop monitors.
- **Semantic HTML:** Built with accessibility and SEO in mind using modern semantic tags.
- **Interactive Elements:** Subtle hover effects and glassmorphism-inspired components for an engaging user experience.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure and metadata.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and advanced transitions.
- **JavaScript (ES6+):** Intersection Observer for scroll animations.
